## 介绍

llm-plus 旨在帮助大模型落地。大模型是当前人工智能最火的领域，但其门槛很高，将开源模型应用到日常生活中是极其复杂的事，本仓库旨在将大模型，与其相关的组件相结合，构造一整套完整的体系：

llm-plus中，plus的含义是对大模型的完善。
